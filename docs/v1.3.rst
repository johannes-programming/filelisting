Introduction
------------

This project creates a ``list`` of all the files under given paths.

Installation
------------

To install filelisting, you can use ``pip``.
Open your terminal and run:

.. code-block:: shell

    pip install filelisting

Typing
------

The typing conforms strictly to ``mypy``.

.. container:: versionadded

    **Added in version 1.3.**

Features
--------

``filelisting.file_generator(*paths: Any) -> Generator[str]``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This generator yields the files under the given path.

``filelisting.file_list(*paths: Any) -> list[str]``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This function returns a ``list`` of the files under the given path.

``filelisting.main``
~~~~~~~~~~~~~~~~~~~~

This click command provides the CLI.

``filelisting.tests.test() -> unittest.TextTestResult``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This project can be tested through its ``test`` function.

.. code-block:: python

    import filelisting.tests
    filelisting.tests.test()
